# Game & Top Up

## Game apa saja yang tersedia di Sultan Top Up?

Kami menyediakan top up untuk berbagai game populer seperti Mobile Legends, Free Fire, PUBG Mobile, Valorant, Genshin Impact, Call of Duty Mobile, dan masih banyak lagi. Daftar game lengkap bisa kamu lihat di halaman utama.

## Berapa lama proses top up?

Proses top up biasanya selesai dalam 1-5 menit setelah pembayaran dikonfirmasi. Pada jam sibuk atau gangguan server game, proses bisa memakan waktu lebih lama, tetapi kami selalu berusaha menyelesaikan secepat mungkin.

## Apa yang harus saya lakukan jika top up tidak masuk?

Jika top up belum masuk setelah 15 menit, cek kembali User ID dan Zone ID yang kamu masukkan. Jika data sudah benar, segera hubungi customer service kami melalui WhatsApp atau halaman Lacak Pesanan dengan menyertakan nomor pesanan.

## Apakah ada batasan jumlah top up?

Tidak ada batasan. Kamu bisa top up sesuai kebutuhan. Untuk nominal besar, sistem kami tetap memproses dengan aman dan cepat.

## Apakah saya bisa top up untuk teman?

Tentu. Kamu tinggal memasukkan User ID dan Zone ID teman kamu saat melakukan pemesanan. Pastikan data yang dimasukkan sudah benar.
